# Upright-car
12th freescale camera upright car project

Technical Documentation by Wyman

NXP Up-Right Car

NXP光电直立平衡车技术文档

一、目的：
	用于恩智浦智能车竞赛

二、环境：  
	MCU：K60 Cortex-M4
	Camera：SONY CCD
	Gyro：ENC-03
	Acce：MMA7361
	Model：E
	Encoder：512
	Bluetooth
	Motor Driver：Piao's PCB
	Others: Piao's PCB

三、命名规则：
	1.函数名用大写首字母的驼峰命名法(没有下划线，下同)；
	2.变量名用小写首字母的驼峰命名法；
	3.宏定义用由下划线隔开的大写单词拼接；
	4.用到库的函数名字不改，方便他人移植。
